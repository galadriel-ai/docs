Here's a list of main features supported natively by the teeML.

We are constantly adding to this list. If you need a specific model or a tool, please create a feature request in [our Github repo](https://github.com/galadriel-ai/contracts/issues/new?assignees=&labels=enhancement&projects=&template=feature-request.md&title=) and join Discord to discuss!

<Tip>
    Try out and see teeML in action at [teeml.galadriel.com](https://teeml.galadriel.com/).
</Tip>


### LLMs

| Provider                                          | Description                                           |
| ------------------------------------------------- | ----------------------------------------------------- |
| [OpenAI](https://platform.openai.com/docs/models) | Models supported: GPT-4-turbo, GPT-3.5-turbo         |
| [Groq](https://console.groq.com/docs/models)      | Models supported: Llama3-8B, Llama3-70B, Mixtral-8x7B, Gemma-7B |
| [Anthropic](https://docs.anthropic.com/en/docs/about-claude/models)      | Models supported: Claude-3.5-Sonnet, Claude-3-Opus, Claude-3-Sonnet, Claude-3-Haiku, Claude-2.1, claude-2.0, claude-instant-1.2 |

### Multimodal

| Provider                                          | Description                                           |
| ------------------------------------------------- | ----------------------------------------------------- |
| [OpenAI](https://platform.openai.com/docs/models) | Models supported: GPT-4o, GPT-4-turbo vision        |

### Text-to-image models

| Provider                                                 | Description                 |
| -------------------------------------------------------- | --------------------------- |
| [OpenAI](https://platform.openai.com/docs/models/dall-e) | Models supported: DALL·E 3 |


### Tools

| Tool          | Description                                |
| ------------- | ------------------------------------------ |
| Google search | Search the web using Google, via [Serper API](https://serper.dev/) |
| Code interpreter | Execute Python code using the [Bearly code interpreter API](https://python.langchain.com/docs/integrations/tools/bearly) |
| Image generation | Enable LLMs to generate images with [DALL·E 3](https://platform.openai.com/docs/guides/images/usage) |

<Tip>
    Currently tools are only available with OpenAI and Anthropic models.
</Tip>

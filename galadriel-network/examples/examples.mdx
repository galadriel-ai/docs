title: Examples
description: List of agent examples built with Galadriel
---

- [DeFAI Trading Agent](https://github.com/galadriel-ai/galadriel/tree/main/examples/trading) - an agent which fetches trending coins, gets market data, does analysis and swaps token.
- [Payments Agent](https://github.com/galadriel-ai/galadriel/tree/main/examples/trading) - a research agent which receives SOL payments from users before executing tasks.
- [Discord Agent](https://github.com/galadriel-ai/galadriel/tree/main/examples/trading) - an agent which simulates Elon Musk's personality, capable of interacting with users in a Discord server
- [Telegram Agent](https://github.com/galadriel-ai/galadriel/tree/main/examples/trading) - an agent which simulates Elon Musk's personality, capable of interacting with users in a Telegram chat
- [Deterministic Agent](https://github.com/galadriel-ai/galadriel/tree/main/examples/trading) - an example of deterministic Twitter "agent" that posts on twitter every 3 hours
